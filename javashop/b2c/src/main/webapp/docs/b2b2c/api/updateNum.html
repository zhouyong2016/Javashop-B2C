更新购物车商品数量

API地址：

/api/shop/cart!updateNum.do?cartid=2&num=1&productid=30

输入参数

名称 类型 是否必须 示例值 默认值 描述
cartid int 必须 1 要更新的购物车项ID
num int 必须 1 更新购物剩余的商品数量
productid int 必须 30 库存表ID

返回结果

名称 类型 是否必须 默认值 描述
result String 必须 如果等于“0”，更新错误，如果等于“1”，那么表示更新成功
