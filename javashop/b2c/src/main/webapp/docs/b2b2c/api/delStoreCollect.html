删除收藏店铺

API地址：

/api/b2b2c/storeCollect!del.do?store_id=1& celloct_id=1

输入参数

名称 类型 是否必须 示例值 默认值 描述
store_id int 是 1 店铺id
celloct_id int 是 1 收藏店铺id

返回结果

名称 类型 是否必须 默认值 描述
result JSON 必须 如果等于“0”，删除失败，如果等于“1”，那么表示删除成功
