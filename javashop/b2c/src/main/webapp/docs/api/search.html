搜索商品

API地址：

/api/shop/goods!search.do?catid=2&brandid=1

输入参数

名称 类型 是否必须 示例值 默认值 描述
catid int 分类id,如果不输入，则搜索全部的分类下的商品
brandid int 品牌id，如果不输入，是搜索全部的品牌下的商品
keyword String 搜索关键字，会搜索商品名称和商品编号

返回结果

名称 类型 是否必须 默认值 描述
result List<Goods> 必须 如果等于“0”，搜索错误，如果等于“1”，那么表示搜索成功
