购物车API

提供了购物车的添加商品（货品）、添加数量、统计价格、删除商品（货品）的功能

API列表

  • addProduct
    将一个货品添加到购物车
  • addGoods
    将一个商品添加到购物车
  • getCartData
    获取购物车数据
  • delete
    删除购物车一项
  • updateNum
    更新购物车数量
  • getTotal
    统计购物车总价信息

数据结构列表

  • CartItem
    购物车条目
  • OrderPrice
    订单价格信息
