搜索会员(要求管理员身份)

API地址：

/api/shop/member!search.do

输入参数

名称 类型 是否必须 示例值 默认值 描述
lvid int admin 会员级别id，如果为空则搜索全部会员级别
keyword String 123456 搜索关键字,可为空

返回结果

名称 类型 是否必须 默认值 描述
result int 必须 为1表示搜索成功，0表示失败
data member 会员列表
