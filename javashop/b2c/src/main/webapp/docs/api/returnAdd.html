returnAdd 对会员购买商品后商品进行退货、换货API

  • 输入参数
  • 返回结果
  • 返回示例
  • API工具

对会员购买商品后商品进行退货、换货。此API必须在会员登录后才可使用。

API地址：

/api/shop/returnorder!returnAdd.do?type=1&ordersn=20140220102237&goodsns=0018914288

输入参数

名称 类型 是否必须 示例值 默认值 描述
type Integer 必须 1 退货、换货类型
ordersn String 必须 20140220102237 订单号
goodsns String 必须 商品货号 订单中的商品货号
applyreason String 选填 退货、换货原因（描述）

返回结果

名称 类型 是否必须 默认值 描述
result int 必须 1 如果等于“0”，退货、换货错误，如果等于“1”，退货、换货错误
message String 返回信息
