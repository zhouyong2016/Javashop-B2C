根据缺货列表的ID,删除当前会员对商品的缺货登记列表。此API必须在会员登录后才可使用。

API地址：

/api/shop/gnotify!gnotifyDel.do?gnotify_id=1

输入参数

名称 类型 是否必须 示例值 默认值 描述
gnotify_id Integer 必须 1 缺货商品列表的id

返回结果

名称 类型 是否必须 默认值 描述
result int 必须 1 如果等于“0”，删除失败，如果等于“1”，删除成功
message String 返回信息
