输入参数

名称 类型 描述
lv_id Integer 评论会员级别ID
levelname String 评论会员级别名称
sex int 评论会员性别，0女，1男
uname String 评论会员会员名
nickname String 评论会员昵称
face String 评论会员头像
comment_id int 评论id
goods_id int 商品id
member_id Integer 会员id
content String 评论内容
img String 评论图片
dateline long 发表时间
ip String 会员的ip
reply String 管理员的回复
replytime long 回复时间(毫秒数)
status int 状态(1为显示，2为隐藏)
type int 类型（1为评论，2为咨询）
replystatus int 回复状态1为已回复，0为未回复
grade int 评分
